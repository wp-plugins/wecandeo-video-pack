Manage and embed videos hosted on Wecandeo.com.


== Description ==

With this plug-in, you can embed to posts and pages the videos that is uploaded in VIDEO PACK. It also can directly upload videos easily. In order to use the plug-in, is required WECANDEO - VIDEO PACK account and API Key.
이 플러그인을 통해 VIDEO PACK에 곧바로 동영상을 업로드하거나 미리 업로드해 둔 동영상과 플레이어를 글과 페이지에 쉽게 Embed할 수 있습니다. 플러그인 사용을 위해서는 WECANDEO - VIDEO PACK 계정과 API Key가 필요합니다.


== Installation ==
Installing should be very easy.
설치는 매우 쉽게 하실 수 있습니다.

1. Upload `wecandeo.zip` into your site plugins directory (typically `/wp-content/plugins/`)
2. Unzip the `wecandeo.zip` file
3. Activate the plugin through the 'Plugins' menu in WordPress
4. http://www.wecandeo.com  the API key (available at my page) and copy it
5. WordPress admin page 'wecandeo' API key to enter the menu and save it.

1. 사이트의 플로그인 디렉토리에 `wecandeo.zip`를 업로드 합니다.(일반적으로 `/wp-content/plugins/`)
2. `wecandeo.zip`압축을 해제 합니다.
3. 워드프레스 관리자페이지에서 'Plugins'메뉴를 통해 wecandeo 플러그인을 활성화합니다. 
4. http://www.wecandeo.com  에서 발급받은 API키(마이페이지에서 확인 가능)를 복사
5. 워드프레스 관리자페이지에서 'wecandeo'메뉴를 통해 API key를 입력 후 저장합니다.
